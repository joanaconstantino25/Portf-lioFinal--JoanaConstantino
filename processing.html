Processing Experiments

Overview

This section presents a selection of creative coding experiments developed with Processing and p5.js. The works explore visual composition, movement, interaction and generative graphics through code.

Focus

The exercises explore basic programming concepts such as shapes, colour, repetition, movement, coordinates and interaction.

Tools

Processing, p5.js, HTML, CSS and JavaScript were used to create and present visual experiments for the web.

Purpose

The goal is to understand how code can become a visual tool for experimentation, composition and digital communication.
